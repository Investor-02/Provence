# Provence

Сайт для банкетного зала «Прованс» в Омске.

В проекте:
- главная страница
- страница зала «Марсель»
- страница зала «Париж»
- галерея
- контакты

## Структура

- `index.html` — главная страница
- `marsel.html` — зал «Марсель»
- `paris.html` — зал «Париж»
- `gallery.html` — галерея
- `contacts.html` — контакты
- `assets/css/styles.css` — стили сайта
- `assets/js/main.js` — интерактивность, меню, модальные окна, галерея
- `assets/images/` — рабочие изображения для сайта

## Локальный запуск

Из папки проекта:

```bash
python -m http.server 4173
```

После этого сайт откроется по адресу:

```text
http://127.0.0.1:4173/index.html
```

Если телефон и компьютер в одной Wi-Fi сети, сайт можно открыть с телефона по локальному IP компьютера:

```text
http://<ваш-ip>:4173/index.html
```
